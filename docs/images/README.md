# 文档图片目录

此目录用于存放 README 和其他文档中使用的图片资源。

## 推荐的图片

在开源发布前，建议添加以下图片：

### logo.png
- 应用 Logo 图标
- 建议尺寸：120x120 像素
- 格式：PNG（透明背景）

### screenshot-triggers.png
- 触发器管理页面截图
- 建议宽度：800-1200 像素

### screenshot-users.png
- 用户管理页面截图
- 建议宽度：800-1200 像素

### screenshot-settings.png
- 系统设置页面截图
- 建议宽度：800-1200 像素

## 截图建议

1. 使用应用的正常窗口大小
2. 确保界面内容完整可见
3. 可以添加一些示例数据使界面更丰富
4. 建议使用 PNG 格式保证清晰度
